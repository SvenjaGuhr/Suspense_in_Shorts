# Suspense in Shorts

This repository documents the results of the URAP project "Measuring Suspense in Fiction" at UC Berkeley, summer term 2026.
<br>

## Suspense Lense
Guhr, Svenja. 2026. SuspenseLens. GitHub Repository. https://github.com/SvenjaGuhr/SuspenseLens. 
<br>
--> A browser-based tool for theory-driven suspense annotation of literary texts

SuspenseLens is a browser-based tool that uses a locally-run Qwen3-4B language model to annotate literary texts sentence-by-sentence for suspense across four theoretical perspectives, visualising results as a colour-coded heatmap and evaluating them against human gold standards.

SuspenseLens lets you load any short story sentence-by-sentence, run it through a local AI model (default: locally-run Qwen3-4B), and receive sentence-level suspense annotations (levels 0; 1-5) from 4 distinct theoretical perspectives: <br>
1) Uncertainty-based Suspense
2) Partial Uncertainty
3) Desire-Frustration (No Uncertainty)
4) Casual Reader <br>
The annotations are displayed as a colour-coded heatmap with colored underlines to indicate the theory. The model performance is evaluated against human gold standard annotations of 15 manually annotated short stories.
